### 🤖 [Punji](https://github.com/ranjulbn20/punji) — Autonomous Personal Finance Agent

> *"What if your portfolio had a devil's advocate?"*

A 9-agent LangGraph system that manages your investments like a hedge fund desk — minus the ₹2Cr minimum.

| Agent | Role |
|-------|------|
| 🧠 Orchestrator | Routes intent, coordinates agent state graph |
| 📊 Portfolio Analyser | XIRR returns, drift detection, rebalancing signals |
| 🌐 Market Intelligence | Real-time signals via yfinance + MFAPI |
| 😈 Devil's Advocate | Adversarially critiques every recommendation |
| 🎲 Goal Tracker | Monte Carlo projections for financial goals |
| 🔍 Concentration Risk | Cross-instrument hidden exposure via AMFI compositions |

**Stack:** LangGraph · LangChain · Gemini / Vertex AI · Qdrant · GCP Cloud Run · Cloud SQL · Redis Streams
